# 📊 Data Analytics Dashboard Project

## 🚀 Overview
This project showcases an interactive Power BI dashboard designed to analyze and visualize key business insights. It helps users explore trends, patterns, and performance metrics through intuitive visualizations.

---

## 📁 Files Included
- Dashboard.pbix → Main Power BI dashboard file.  
- Insights.pbix → Additional insights and analysis.  
- Dashboard.png → Preview of the dashboard.  
- Preview.png → Photo of Insights.
- Exam.SQL → SQL File.

---

## 📌 Features
- 📈 Interactive data visualizations  
- 🔍 Drill-down analysis for deeper insights  
- 📊 Key performance indicators (KPIs)  
- 🧭 User-friendly dashboard layout  
- ⚡ Real-time or dynamic filtering (if connected to live data)
